HASH TABLE GAME

INSTRUCTIONS

This is a tool to help learn about hash tables. There are three kinds of hashing that are commonly used: linear probing, separate chaining and double hashing. This version of this 'game' uses linear probing.

INSERT is the number that needs to be added to the hash table during this round. To insert it, use the FUNCTION to determine which index it should go in. If that index is filled already, then linearly probe through the hashtable until you find an open spot. Click on the green boxes to add a number to the table.
